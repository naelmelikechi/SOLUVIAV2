1 Principes

Cette spec enrichit la page detail projet /projets/[ref] (spec 02) avec 6 volets de suivi. L'objectif est de maximiser les indicateurs en pourcentage pour chaque projet.

📈

6 volets de suivi

Financier, Qualite, Pedagogie, Reussite, Financement, Abandons. Chaque volet a ses indicateurs en %.

💰

Rentabilite

Indicateur de rentabilite par projet. Formule a definir.

🔄

Remplace la spec 02

Les sections Finance, Temps et Qualite de la page detail projet (spec 02) sont remplacees par ces volets enrichis.

RegleDetail
Page/projets/[ref] — meme URL que la spec 02. Cette spec enrichit le contenu de la page detail.
Backup CDPChamp backup CDP (chef de projet referent si le CDP principal est absent). Deja defini en spec 02.
PermissionsCDP voit ses projets (RLS). Admin voit tout. Lecture seule — les donnees viennent des autres modules.
ObjectifMaximiser les indicateurs en %. Chaque volet doit permettre d'identifier rapidement les points faibles du projet.

5 Volets de performance — Indicateurs %

6 volets d'indicateurs qualitatifs. L'objectif est de maximiser chaque pourcentage. Les formules exactes seront definies ulterieurement.

⚠

A definir — Les formules de calcul, les sources de donnees et les seuils de ces 6 indicateurs seront specifies ulterieurement. Les emplacements sont reserves dans l'interface. Pour chaque indicateur, il faudra definir : (1) la formule exacte, (2) le "relatif a quoi" (denominateur du %), (3) les seuils de couleur (vert/orange/rouge), (4) la source de donnees.

Principes de conception

PrincipeDetail
Relatif a quoiChaque % doit avoir un denominateur clair. Ex: Qualite = livrables faits / livrables totaux. Abandons = resilies / total contrats. Le "relatif a quoi" est le point le plus important a definir.
ObjectifTous les indicateurs doivent tendre vers 100% (sauf Abandons qui doit tendre vers 0%).
Seuils de couleurVert = bon (≥ 80%). Orange = attention (50-79%). Rouge = critique (< 50%). Seuils ajustables par indicateur.
ConfigurableChaque volet est masquable. L'utilisateur choisit les indicateurs affiches.
Ajout sans refonteL'ajout d'un indicateur ne necessite pas de refonte de l'interface — juste la configuration de la formule et de la source.
