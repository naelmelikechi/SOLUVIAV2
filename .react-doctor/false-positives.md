## react-doctor/iframe-missing-sandbox

- `components/qualiopi/notice-iframe.tsx` — combines `allow-scripts` with `allow-same-origin` intentionally. Iframe loads `https://quavia.eduvia.app` (different origin from SOLUVIA), so `allow-same-origin` only re-grants the iframe access to its own origin's cookies, not SOLUVIA's. The dangerous combination is when both flags are granted on an iframe loading content from the parent's own origin — not this case.
