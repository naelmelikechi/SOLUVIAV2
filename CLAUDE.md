# SOLUVIA

## Project Overview

Strategic control layer for French training organizations (organismes de formation).
Reads operational data from Eduvia, manages internal processes (time tracking, quality, invoicing),
and pushes invoices to Odoo. Detailed specs in `specs/` (00-09).

## Tech Stack

- Next.js 16 (App Router) with TypeScript
- TailwindCSS 4 + shadcn/ui (base-ui)
- Supabase (PostgreSQL, Auth, RLS, Realtime, Edge Functions, Storage)
- Deployed on Vercel

## Commands

- `npm run dev` — Start dev server (Turbopack)
- `npm run build` — Production build
- `npm run lint` — ESLint
- `npx supabase start` — Start local Supabase
- `npx supabase db push` — Apply migrations
- `npx supabase gen types typescript --local > types/database.ts` — Regenerate types

## Architecture

- App Router with route groups: `(auth)` for login, `(dashboard)` for main app
- Next.js 16: uses `proxy.ts` (not `middleware.ts`) for auth middleware
- Data fetching in Server Components; `'use client'` only for interactivity
- Supabase RLS enforces permissions at DB level (admin sees all, CDP sees own projects)
- shadcn/ui uses base-ui (not radix) — no `asChild` prop, no `delayDuration` on Tooltip

## Database

- Table names: French, snake_case (projets, contrats, saisies_temps, factures)
- Primary key: `id UUID DEFAULT gen_random_uuid()`
- Business refs auto-generated by triggers: project `0042-DUP-APP`, contract `CTR-00187`, invoice `FAC-DUP-0012`
- Timestamps: `created_at`, `updated_at` (auto via trigger)
- Soft delete: `archive BOOLEAN DEFAULT false`
- Invoice numbering: gapless via max+1 (French legal requirement) — never DELETE invoices

## Roles

- `admin`: Full R+W access everywhere
- `cdp` (Chef de Projet): Filtered by `cdp_id = auth.uid() OR backup_cdp_id = auth.uid()`
- CDP cannot access `/admin/parametres` (hidden from sidebar)

## Conventions

- Components: PascalCase files (`Sidebar.tsx`)
- Utilities: camelCase files (`formatters.ts`)
- All French domain terms kept in French (projet, contrat, facture, qualite, temps)
- UI text: French
- All data tables use the shared `DataTable` component from `components/shared/data-table/`
- Auto-save pattern: 2s debounce via `useDebounce` hook (for time tracking)
- CRON routes protected by `CRON_SECRET` bearer token
- Supabase Realtime for sidebar badge counts (notifications, overdue invoices, time not logged)
